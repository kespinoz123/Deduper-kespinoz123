------------------------------------------------------------------------------------------------------------         
                                    Commands Used After Completing Deduper
------------------------------------------------------------------------------------------------------------ 

--------- Header Count ---------
> cat C1_SE_uniqAlign.deduped_.sam | grep -E "^@" | wc -l
output: 64

--------- Original Reads --------- 
> cat sorted_C1_SE_uniqAlign.sam | grep -v "^@" | wc -l
output: 18186410

--------- Unique Reads --------- 
> cat C1_SE_uniqAlign.deduped_.sam | grep -v "^@" | wc -l
output: 13729938


--------- Deduplicates --------- 
# of original reads - # of unique reads = # of deduplicates
18186410-13729938= 4456472

--------- Unique Reads Per Chromsome --------- 
> cat C1_SE_uniqAlign_deduped_.sam | grep -v "^@" | cut -f 3 | sort |  uniq -c > unique_reads_per_chromosome.txt
